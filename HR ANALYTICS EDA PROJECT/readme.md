# 📊 Exploratory Data Analysis (EDA) on Sales Dataset

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a sales dataset to understand business performance, identify trends, and discover meaningful insights using Python.

The analysis includes data cleaning, statistical summaries, and visualizations to explore sales patterns across different categories, regions, customers, and time periods.

---

## 🛠️ Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
